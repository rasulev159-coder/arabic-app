# ТЗ: Реструктуризация UX — чёткий путь от нуля до Корана

## Цель
Переделать навигацию и главный экран так, чтобы новый пользователь сразу понимал:
1. С чего начать
2. Где он сейчас
3. Что дальше
4. Сколько осталось

Методология: Муаллим Сони + Spaced Repetition + Gamification.

---

### Визуальный стиль roadmap:

- Вертикальная линия соединяет этапы (как timeline)
- Пройденные этапы: золотая линия + галочка
- Текущий этап: пульсирующий индикатор + прогресс-бар
- Заблокированные: серые + иконка замка
- При клике на этап — раскрывается список уроков
- Каждый урок: чекбокс (пройден/нет) + название + кнопка "Начать"

## 8. Иконки — замена эмодзи на Lucide

### Во ВСЁМ приложении заменить эмодзи на Lucide React иконки:

```typescript
// БЫЛО (Dashboard, Nav, etc.)
{ icon: '🏠' }
{ icon: '📖' }
{ icon: '⚡' }

// СТАЛО
import { Home, BookOpen, Zap, Brain, Search, Pen, Volume2, Target } from 'lucide-react';
{ icon: Home }
{ icon: BookOpen }
{ icon: Zap }
```

Маппинг эмодзи → Lucide:
- 🏠 → `Home`
- 📖 → `BookOpen`
- 📈 → `TrendingUp`
- 🏆 → `Trophy`
- 🥇 → `Medal`
- ⚙️ → `Settings`
- 📇 → `Layers` (flashcards)
- 🎯 → `Target` (quiz)
- ⚡ → `Zap` (speed)
- 🌩️ → `CloudLightning` (lightning)
- 🧠 → `Brain` (memory)
- 🔊 → `Volume2` (listen)
- 🔍 → `Search` (find)
- ✍️ → `PenTool` (write)
- 💬 → `MessageCircle` (chat)
- 👤 → `User` (profile)
- 📍 → `MapPin` (learn path)
- 🎮 → `Gamepad2` (practice)
- 🔒 → `Lock` (locked stage)
