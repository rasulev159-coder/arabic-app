# Business Model Canvas — Arab Alifbosi

## 1. Customer Segments

### Основной сегмент:
- Узбекистанцы 16-45 лет, которые хотят научиться читать арабский с нуля
- Мотивация: чтение Корана + общее развитие

### Вторичный сегмент:
- Учителя арабского / имамы — хотят инструмент для учеников
- Родители — хотят чтобы дети учились

### Будущий сегмент:
- Мусульмане из Казахстана, Кыргызстана, Таджикистана, России

## 2. Value Propositions

| Для ученика | Для учителя |

## 9. Cost Structure

### Сейчас (полностью бесплатно):
| Статья | Стоимость |
|--------|----------|
| Vercel Hobby | $0 |
| Neon Free | $0 |
| Upstash Free | $0 |
| Домен (нужен) | ~$10/год |
| **Итого** | **~$10/год** |

### При масштабировании (1,000+ MAU):
| Статья | Стоимость |
|--------|----------|
| Vercel Pro | $20/мес |
| Neon Launch | $19/мес |
| Upstash Pay-as-you-go | ~$5/мес |
| Домен | ~$1/мес |
| **Итого** | **~$45/мес** |

### Breakeven:
- При подписке $1.99/мес: нужно **23 платящих пользователя**
- При 1,000 MAU и 5% конверсии: **50 платящих** = $100/мес (прибыль $55)
