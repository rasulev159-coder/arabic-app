# TAM / SAM / SOM — Arab Alifbosi

## TAM (Total Addressable Market)
**Все, кто хочет изучить арабский алфавит в мире**

- Мусульмане, не умеющие читать Коран: ~600 млн человек (по оценкам, ~35% из 1.8 млрд мусульман не читают по-арабски)
- Студенты арабского языка: ~30 млн
- **TAM: ~630 млн человек**
- При средней выручке $5/пользователь/год: **~$3.15 млрд**

## SAM (Serviceable Addressable Market)
**Русско/узбекскоязычные мусульмане с доступом к интернету**

### Узбекистан:
- Население: 36 млн
- Мусульмане: ~33 млн (92%)
- Не умеют читать арабский: ~25 млн (оценка ~75%)
- С доступом к смартфону: ~20 млн
- **Узбекистан SAM: ~20 млн**

### Другие страны СНГ (рус/узб язык):
- Казахстан: ~5 млн мусульман со смартфонами
- Кыргызстан: ~3 млн
- Таджикистан: ~4 млн
- Россия (мусульмане): ~8 млн
- Другие: ~2 млн
- **СНГ SAM: ~22 млн**

### Итого SAM:
- **~42 млн потенциальных пользователей**
- При конверсии в freemium ($3/год): **~$126 млн**

## SOM (Serviceable Obtainable Market)
**Реалистичная цель на первый год**

### Ограничения:
- Нет маркетингового бюджета
- Один разработчик
- Только органический рост (сарафан, соцсети, мечети)

### Расчёт:
- Узбекистан, через Telegram/Instagram + мечети
- Целевая аудитория в первый год: молодёжь 16-35 в крупных городах
- Охват: Ташкент, Самарканд, Бухара, Фергана = ~5 млн смартфон-пользователей
- Конверсия в установку: 0.1% = **5,000 пользователей**
- Активных (MAU): 30% = **1,500 MAU**
- Платящих (если монетизация): 3% = **45 человек × $3/мес = ~$135/мес**

### SOM Year 1:
- **5,000 зарегистрированных**
- **1,500 активных в месяц**
- **$1,600/год выручка** (если запустить подписку)

### SOM Year 2 (с контент-маркетингом):
- **25,000 зарегистрированных**
- **7,500 MAU**
- **$8,000/год выручка**

## Ключевые инсайты

1. **Рынок огромный** — 42 млн в СНГ, почти никто не обслужен качественно
2. **Конкуренция слабая** — AlifBee дорогой, локальные приложения убогие
3. **Монетизация рано** — сначала набрать базу, потом freemium
4. **Рост через мечети** — один имам = 200-500 пользователей
5. **Вирусный потенциал** — "вызов друга" + share cards = органический рост
