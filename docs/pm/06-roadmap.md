## Phase 2: PRODUCT-MARKET FIT (Май — Июнь 2026)
**Цель: Retention D7 > 25%. 300 активных пользователей.**

| Приоритет | Задача | Описание |
|-----------|--------|----------|
| P0 | Реальные аудио-файлы | Записать/найти озвучку 28 букв носителем |
| P0 | Onboarding flow | Первый запуск: выбор языка → 3-минутный тур |
| P1 | Учебник расширение | Добавить главы: слоги, простые слова |
| P1 | Улучшить дизайн | Нанять дизайнера или использовать референсы |
| P2 | Оффлайн-режим | Service Worker кэширует весь контент |
| P2 | Email-уведомления | Подключить Resend для реальной отправки |

**Метрики:**
- 300 зарегистрированных
- D7 retention > 25%
- 3+ сессий/пользователь/неделю
